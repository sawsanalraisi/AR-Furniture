# AR-FurnitureStore
Experience retail in Augmented Reality. Place and manipulte 3D furniture in real space, for a personalized experience.

### Functionality -
* Rotate
* Move
* Color
* Scale

### Build Using -
Unity 2018.1.0f2 (64-bit) <br />
ARCore 1.5.0
